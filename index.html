Редактор аудиоплеера для Genially v3

Сначала редактор вытаскивает реальные mp3-ссылки, потом встраивает их в готовый iframe. Фон у плеера прозрачный.

Важно: ссылка https://chekanaska.github.io/audioplayer/ — это не “папка” со списком, а страница. В этой версии редактор понимает GitHub Pages и сам ищет файлы в репозитории.
